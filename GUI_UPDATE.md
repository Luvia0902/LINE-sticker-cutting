## 🎯 精確規格支援

### 貼圖集標準規格
根據您的需求，系統現在支援精確的貼圖集規格：

```
整體畫幅：2560 × 1664 px（橫向矩形）
結構佈局：4 橫排 (Rows) × 5 直欄 (Columns)
每格尺寸：512 × 416 px（自動計算）
總貼圖數：20 張
```

```bash

```

**優點**：
- 視覺化操作，適合新手

- 進度追蹤，狀態清晰
- 一鍵開啟結果目錄

### 方法 2：命令列版本
```bash

```

**優點**：
- 適合批次處理
- 可腳本化自動執行
- 適合整合到工作流程

```bash

```

**優點**：
- 統一的任務管理
- 可擴展更多步驟
- 符合 Agent 架構

## 🎨 GUI 介面說明

### 主視窗佈局
```
┌──────────────────────────────────────────────────────┐
│          🎨 LINE 貼圖切割工具 v1.0                    │
│        精確切割貼圖集為獨立貼圖                        │
├─────────────────────┬────────────────────────────────┤
│ [左側控制面板]      │  [右側預覽區]                  │
│                     │                                │
│ 📂 輸入文件         │  🖼️ 圖片預覽                   │
│ 💾 輸出目錄         │  ┌──────────────────────┐      │
│ ⚙️ 規格設定         │  │                      │      │
│                     │  │   [預覽圖]           │      │
│ [🚀 開始切割]       │  │   [紅色網格線]       │      │
│ [📁 開啟輸出目錄]   │  │                      │      │
│                     │  └──────────────────────┘      │
├─────────────────────┴────────────────────────────────┤
│ 狀態   [進度條 ████████░░ 80%]                       │
└──────────────────────────────────────────────────────┘
```

### 關鍵組件
1. **輸入文件選擇**：瀏覽並選擇貼圖集圖片
2. **輸出目錄設定**：指定輸出位置（預設 `.tmp/output/stickers`）
3. **規格設定區**：
   - 標準規格開關（2560×1664，4×5）
   - 貼圖集尺寸設定
   - 行列數設定
   - 自動計算結果顯示
4. **預覽區**：即時圖片預覽 + 網格線覆蓋
5. **操作按鈕**：開始切割、開啟輸出目錄
6. **狀態欄**：狀態文字 + 進度條

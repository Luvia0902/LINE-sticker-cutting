# 🎯 格子內圖案縮放功能

## ✅ 功能說明（最終版）

**核心概念**：貼圖集外框保持不變（2560×1664），只縮放每個格子內的圖案內容

---

## 📊 工作原理

### 貼圖集結構（不變）
```
2560 × 1664 px 貼圖集
┌─────┬─────┬─────┬─────┬─────┐
│512× │512× │512× │512× │512× │
│416  │416  │416  │416  │416  │ 4 橫排
├─────┼─────┼─────┼─────┼─────┤
│     │     │     │     │     │
├─────┼─────┼─────┼─────┼─────┤ × 
│     │     │     │     │     │
├─────┼─────┼─────┼─────┼─────┤ 5 直欄
│     │     │     │     │     │
└─────┴─────┴─────┴─────┴─────┘
    ↑ 每個格子 512×416 px（不變）
```

### 圖案縮放（可調整）

#### 50% - 圖案縮小，留白增加
```
每個格子 512×416 px
┌───────────────┐
│               │
│   ┌───────┐   │  圖案：256×208 px
│   │       │   │  留白：增加
│   │ 圖案  │   │
│   │       │   │
│   └───────┘   │
│               │
└───────────────┘
```

#### 100% - 標準尺寸
```
每個格子 512×416 px
┌───────────────┐
│┌─────────────┐│  圖案：512×416 px
││             ││  留白：無
││    圖案     ││
││             ││
│└─────────────┘│
└───────────────┘
```

#### 150% - 圖案放大，可能超出
```
每個格子 512×416 px
  ┌─────────────┐
┌─┼─────────────┼─┐  圖案：768×624 px
│ │             │ │  (超出格子範圍)
│ │    圖案     │ │
│ │             │ │
└─┼─────────────┼─┘
  └─────────────┘
```

## 💡 使用場景

### 場景 1：增加留白空間（50-90%）
**用途**：
- 讓貼圖圖案周圍有更多空白
- 避免圖案太靠近邊緣
- 更符合 LINE 貼圖規範

**效果**：
- 圖案縮小
- 自動置中
- 周圍透明留白增加

### 場景 2：標準尺寸（100%）⭐
**用途**：
- 使用原始圖案尺寸
- 最常用設定

**效果**：
- 圖案填滿格子
- 無額外留白

### 場景 3：放大圖案（110-150%）
**用途**：
- 讓主要圖案更醒目
- 可裁切部分邊緣內容

**效果**：
- 圖案放大
- 可能超出格子範圍（會被裁切）

## 🔧 技術實現

### 預覽處理
```python
# 1. 創建新畫布（保持 2560×1664）
scaled_img = Image.new('RGBA', (2560, 1664), transparent)

# 2. 對每個格子處理
for each cell (512×416):
    # 裁切原始格子
    cell = original.crop(cell_area)
    
    # 縮放格子內容
    scaled_cell = cell.resize((512 * percent / 100, 416 * percent / 100))
    
    # 置中貼回
    paste_position = center_of_cell
    scaled_img.paste(scaled_cell, paste_position)

# 3. 整體貼圖集尺寸不變，只有內容縮放
```

### 實際切割處理
```python
# 1. 裁切單個格子
sticker = original.crop((left, top, right, bottom))

# 2. 如果非 100%，縮放內容
if scale_percent != 100:
    # 縮放圖案
    scaled = sticker.resize((width * percent / 100, height * percent / 100))
    
    # 創建透明畫布（保持格子尺寸）
    canvas = Image.new('RGBA', (512, 416), transparent)
    
    # 將縮放後的圖案置中貼上
    canvas.paste(scaled, center_position)
    sticker = canvas

# 3. 添加留白（可選）
# 4. 調整為 LINE 尺寸 (370×320)
# 5. 儲存
```

## 🎯 與留白功能的搭配

### 兩種留白方式

#### 1. 圖案縮放（本功能）
- **調整對象**：格子內的圖案
- **效果**：圖案縮小，周圍留白增加
- **範圍**：50% 到 150%
- **使用**：調整「圖片大小調整」滑桿

#### 2. 留白邊距（原有功能）
- **調整對象**：最終輸出的邊緣
- **效果**：圖案整體縮小 10px，邊緣留白
- **固定**：10px
- **使用**：勾選「添加留白邊距」

#### 組合使用
```
圖案縮放 75% + 留白邊距 10px
= 格子內圖案先縮小到 75%
+ 最終輸出時邊緣再留 10px
= 雙重留白效果
```

## 📊 縮放對照表

| 百分比 | 格子內圖案尺寸 | 效果 | 建議用途 |
|--------|---------------|------|----------|
| 50% | 256×208 px | 大量留白 | 極簡設計 |
| 75% | 384×312 px | 適度留白 | 一般貼圖 |
| **100%** | **512×416 px** | **無額外留白** | **標準** ⭐ |
| 125% | 640×520 px | 圖案放大 | 醒目設計 |
| 150% | 768×624 px | 圖案超出格子 | 裁切效果 |

---

## ⚠️ 重要說明

### 最終輸出尺寸（固定）
無論縮放百分比如何，最終輸出都是：
- **370 × 320 px**（LINE 標準）
- 縮放只影響圖案在格子內的大小
- 不影響最終文件尺寸

### LINE 上架建議
- ✅ **推薦 75-100%**（適度留白，符合規範）
- ⚠️ **50%**（留白過多，圖案可能太小）
- ⚠️ **>100%**（圖案可能被裁切）

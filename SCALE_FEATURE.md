## ⚙️ 技術實現

### 縮放計算
```python
# 基準尺寸
STICKER_SIZE = (370, 320)

# 應用縮放
scale_percent = 100  # 使用者設定的百分比
actual_width = int(370 * scale_percent / 100)
actual_height = int(320 * scale_percent / 100)

# 範例：75% 縮放
# actual_width = int(370 * 75 / 100) = 278
# actual_height = int(320 * 75 / 100) = 240
```

### 即時更新
```python
def update_scale_label(self, value=None):
    """滑桿值改變時立即更新顯示"""
    scale = self.scale_percent.get()
    
    # 更新百分比顯示
    self.scale_value_label.config(text=f"{scale}%")
    
    # 計算並顯示實際尺寸
    actual_w = int(370 * scale / 100)
    actual_h = int(320 * scale / 100)
    self.scale_preview_label.config(
        text=f"➜ 實際輸出：{actual_w}×{actual_h}px"
    )
```

## ⚠️ 注意事項

### LINE 貼圖規格
LINE 官方規格要求：
- **最大**：370 × 320 px
- **建議**：使用 100%（標準尺寸）

如果您要上傳到 LINE Creators Market：
- ✅ 使用 **100%**（符合規格）
- ⚠️ 使用 **>100%** 會超出官方規格
- ✅ 使用 **<100%** 符合規格但較小

### 檔案大小
- 縮放比例越大，檔案越大
- 所有輸出都會自動優化壓縮
- 確保每張貼圖 < 1MB

### 圖片品質
- **50% ~ 100%**：縮小，品質良好
- **100%**：原始品質（推薦）
- **101% ~ 150%**：放大，可能略微模糊

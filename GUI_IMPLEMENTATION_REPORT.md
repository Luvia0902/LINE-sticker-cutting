## 🚀 立即開始使用

### 啟動 GUI
```bash
python execution\sticker_splitter_gui.py
```

### 使用流程
1. **選擇圖片** - 點擊「選擇圖片」
2. **檢查預覽** - 查看圖片和紅色網格線
3. **確認規格** - 預設 2560×1664，4×5（可自訂）
4. **開始切割** - 點擊「🚀 開始切割」
5. **查看結果** - 點擊「📁 開啟輸出目錄」

## 🔄 多種使用方式

### 方式對比

| 方式 | 優點 | 適用場景 | 命令 |
|------|------|----------|------|
| **GUI** | 視覺化、易用、預覽 | 單次處理、新手 | `python execution\sticker_splitter_gui.py` |
| **命令列** | 批次、自動化 | 批次處理、腳本 | `python execution\split_stickers.py -i image.png` |
| **編排器** | 統一管理、可擴展 | 複雜工作流 | `python execution\agent_orchestrator.py --task split-stickers` |

### 選擇建議
- 🟢 **新手/偶爾使用** → GUI
- 🟡 **批次處理** → 命令列
- 🔴 **複雜工作流** → 編排器

## 🎯 更新內容

### 1️⃣ **整體圖片縮放預覽** ✨
**改變**：預覽區顯示整個貼圖集的縮放效果，而不是單張貼圖

**效果**：

**使用方式**：

## 🔧 技術說明

### 整體縮放實現

```python
# 1. 取得縮放百分比
scale_percent = self.scale_percent.get()  # 例如：75

# 2. 應用到整個原圖
scaled_width = int(original_width * 75 / 100)
scaled_height = int(original_height * 75 / 100)
scaled_img = original_img.resize((scaled_width, scaled_height))

# 3. 再縮放以適應畫布（保持長寬比）
preview_img = scale_to_fit_canvas(scaled_img)

# 4. 顯示在預覽區
# 結果：整個貼圖集按比例縮小/放大
```

### 數字輸入綁定

```python
# 創建輸入框
spinbox = ttk.Spinbox(
    from_=50,      # 最小值
    to=150,        # 最大值
    textvariable=self.scale_percent,  # 綁定變數
    width=8
)

# 綁定事件
spinbox.bind('<Return>', update_preview)     # Enter 鍵
spinbox.bind('<FocusOut>', update_preview)   # 失去焦點
```

## ⚠️ 注意事項

### 預覽 vs 實際輸出

**預覽區**：
- 顯示整體貼圖集的縮放效果
- 幫助您視覺化整體大小

**實際輸出**：
- 每張單獨貼圖會調整為設定的尺寸
- 例如 75% → 每張貼圖 278×240 px

### LINE 上架建議

如果要上傳 LINE：
- ✅ **推薦 100%**（370×320，符合官方規格）
- ⚠️ 其他百分比可能不符合 LINE 規格

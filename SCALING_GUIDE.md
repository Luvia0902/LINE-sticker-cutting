## 📦 工作原理

### 固定輸出尺寸
- **所有貼圖檔案 = 370×320 px**（永遠不變）
- 符合 LINE 官方標準規格

### 可變內容大小
- **50%** = 圖片內容縮小，周圍留白增加
- **100%** = 圖片內容標準大小（推薦）
- **150%** = 圖片內容放大，智能適應避免裁切

## ⚙️ 智能適應功能

### 什麼時候啟用？
- 僅當 **縮放 > 100%** 且 **內容會超出框架** 時

### 如何運作？
```
1. 內容放大到 555×480 (150% of 370×320)
2. 檢測：555 > 370 或 480 > 320（超出！）
3. 計算縮放比例：370/555 = 0.667
4. 自動調整內容大小以適應
5. 置中放入 370×320 框架
6. 結果：完整內容，無裁切
```

### 何時不啟用？
- **縮放 ≤ 100%**（縮小不會超出）
- **取消勾選智能適應**（允許裁切）

## 🎯 核心重點

### ✅ 正確理解
```
輸出檔案尺寸 = 370×320 px（固定）
圖片內容大小 = 可調整 50%~150%
框架 = 不變
內容 = 可變
```

### ❌ 錯誤理解
```
輸出檔案尺寸 = 會跟著百分比變化
（這是錯的！檔案永遠 370×320）
```
